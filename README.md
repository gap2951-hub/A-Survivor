# A-Survivor

방향키로만 캐릭터를 조작하고, 사방에서 몬스터가 끊임없이 몰려오는 생존형 안드로이드 게임.

## 게임 소개

- 플레이어는 **방향키(D-pad)** 로만 이동
- 전투는 **자동** — 공격 버튼 없음
- 몬스터는 **사방에서 지속 스폰**
- 얼마나 오래 살아남을 수 있을지가 목표

## 기술 스택

- **언어:** Kotlin
- **UI:** Jetpack Compose
- **minSdk:** 24 / **targetSdk:** 36

## 빌드 방법

1. 저장소 클론
   ```bash
   git clone https://github.com/gap2951-hub/A-Survivor.git
   ```
2. Android Studio에서 프로젝트 열기
3. Run `app`
