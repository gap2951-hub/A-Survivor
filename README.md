# A-Survivor

메이플스토리 스타일의 픽셀아트 사냥터를 배경으로 한 안드로이드 생존형 RPG 게임.
가상 조이스틱으로 이동하고, 자동 공격으로 슬라임을 처치하며 장비를 강화한다.

## 게임 플레이

| 조작 | 설명 |
|------|------|
| 가상 조이스틱 (좌하단) | 캐릭터 이동 |
| 자동 공격 | 공격 범위(120f) 내 최근접 몬스터를 1초마다 자동 공격 |
| 아이템 픽업 | 몬스터 처치 후 드랍된 아이템에 가까이 가면 자동 습득 |
| 강화 | 인벤 → 아이템 꾹 눌러 장갑 슬롯으로 드래그 |

## 주요 시스템

### 몬스터 AI (비선공)
- 평소에는 IDLE 상태로 자리를 지킴
- 플레이어에게 공격받으면 AGGRO 전환 → 플레이어 추적 시작
- 근접 시 1초마다 데미지 5 반격
- 플레이어가 500f 이상 멀어지면 IDLE 복귀

### 픽셀 충돌 시스템
- 맵 이미지 픽셀의 밝기를 실시간 분석
- 나무·풀숲(어두운 픽셀)은 플레이어·몬스터 모두 통과 불가
- 벽 슬라이딩으로 자연스러운 이동 구현

### 주문서 강화 시스템 (메이플스토리 스타일)
- 100% / 60% / 10% 주문서로 장갑 공격력 강화
- 백의 주문서로 실패 횟수 복구
- 강화 실패 시 장비 파괴 가능

### 드랍 시스템
- 슬라임 처치 시 확률에 따라 아이템 바닥에 드랍
- 플레이어가 50f 이내 접근하면 자동 습득

### 스탯 / 레벨 시스템
- 몬스터 처치 EXP → 레벨업 시 SP 획득
- STR / DEX / INT / LUK 자유 배분
